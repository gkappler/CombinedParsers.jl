A fast Trie-based parser for a collection of literal Strings.

Matching any one of many Strings (e.g. names) is slow in PCRE with | alternatives.

julia> using Random

julia> s = [ randstring(10) for _ in 1:1000 ];

julia> re = Regex(join(s,"|"));

julia> using BenchmarkTools

julia> @benchmark match(re,s[end])
BenchmarkTools.Trial:
  memory estimate:  256 bytes
  allocs estimate:  5
  --------------
  minimum time:     1.963 μs (0.00% GC)
  median time:      2.011 μs (0.00% GC)
  mean time:        2.062 μs (0.00% GC)
  maximum time:     4.808 μs (0.00% GC)
  --------------
  samples:          10000
  evals/sample:     10

A optimized state-machine parser will perform much better.

CombinedParsers does not use state machines, but provides fast _iterate implementation for Either{Trie{Char}}, that is constructed with Either(::Vector{<:AbstractString})

julia> using CombinedParsers

julia> pc = Either(s);

julia> @benchmark match(pc,s[end])
BenchmarkTools.Trial:
  memory estimate:  240 bytes
  allocs estimate:  7
  --------------
  minimum time:     175.308 ns (0.00% GC)
  median time:      185.765 ns (0.00% GC)
  mean time:        232.274 ns (18.33% GC)
  maximum time:     11.401 μs (98.17% GC)
  --------------
  samples:          10000
  evals/sample:     746

Note that pc does not capture the result by default with parse.

julia> parse(!pc,s[end])
"umPmQq339r"

For large word collections, PCRE fails

julia> s = [ randstring(10) for _ in 1:10000 ];

julia> re = Regex(join(s,"|"));
ERROR: PCRE compilation error: regular expression is too large at offset 109999

CombinedParsers displays good time complexity.

julia> pc = Either(s);

julia> @benchmark Either(s);

julia> @benchmark match(pc,s[end])
BenchmarkTools.Trial:
  memory estimate:  240 bytes
  allocs estimate:  7
  --------------
  minimum time:     176.498 ns (0.00% GC)
  median time:      187.933 ns (0.00% GC)
  mean time:        229.960 ns (16.26% GC)
  maximum time:     12.006 μs (98.09% GC)
  --------------
  samples:          10000
  evals/sample:     723

Either{Trie{Char}} can be used to perform fast text search in Julia.
