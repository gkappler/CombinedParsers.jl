CombinedParsers.Regexp.ParseMatch â€” Type

Wrapper type for SequenceWithCaptures, providing getindex and getproperty behavior like RegexMatch.

julia> m = match(re"(?<a>so)+ (or)", "soso or")
ParseMatch("soso or", a="so", 2="or")

julia> m[:a]
"so"

julia> m[2]
"or"

julia> m.match, m.captures
("soso or", SubString{String}["so", "or"])
